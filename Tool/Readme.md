## 기능 소개
- 파일 읽는 속도를 개선하기 위해 CSV파일을 bytes파일로 변환하는 툴을 개발했습니다.
- 파일의 내용을 리틀 엔디안 방식으로 변환하여 저장하는 간단한 기능입니다.
- 다수의 시간이 소요되는 프로젝트 파일을 열지 않고 작업이 가능하도록 툴 형태로 작업하였으며 비프로그래머도 사용할 수 있습니다.

#### Create Test CSV 버튼
![image](https://github.com/user-attachments/assets/74010056-21aa-4a44-975b-0bb88dc6ab97)

- 약 300mb 용량의 기능 테스트용 CSV 파일을 생성합니다.
- 가장 첫 줄에는 데이터 타입을 저장합니다. 임의의 숫자는 정확하게 자료형을 확인할 수 없기 때문에 첫 줄에 명시적으로 데이터 타입을 나타냅니다.
- 툴이 읽는 모든 CSV 파일은 첫 줄에 데이터 타입을 나타내야합니다.

#### 파일 경로 텍스트 박스
- bytes 파일로 생성하고자 하는 CSV 파일의 파일 경로를 적는 칸입니다.

#### Click 버튼
- 파일 경로에 있는 모든 CSV 파일을 직렬화 하여 bytes 파일을 생성합니다.
- 첫 라인을 읽어 데이터 타입을 사전에 확인하고 그 이후의 데이터를 직렬화 하는데 사용합니다.
